Отделы:
ГБУ “КУРТАМЫШСКАЯ ЦРБ” разделяется на отделения, занимающиеся оказанием различных медицинских услуг населению.
  • Гинекологическе отделение.
  • Терапевтическое отделение.
  • Акушерское отделение.
  • Реанимационное отделение.
  • Поликлиника детская.
Организационная структура предприятия.
I. Административно-управленческий аппарат
  • Главный врач
  • Заместитель главного врача по медицинской части
  • Заместитель главного врача по организационно-методической работе
  • Заместитель главного врача по поликлинике
  • Заместитель главного врача по клинико-экспертной работе
  • Главная медицинская сестра
  • Главный бухгалтер
  • Заместитель главного врача по экономическим вопросам
  • Заместитель главного врача по административно-хозяйственной части
  • Специалист по кадрам
  • Специалист по гражданской обороне и мобилизационной работе
